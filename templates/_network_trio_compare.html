Four views of the same data

These four tools ask different questions of the citation graph. Each surfaces a different kind of relationship; the same article can sit at the center of one view and the periphery of another.

  • {% if current == "citation" %}Citation Network{% else %}Citation Network{% endif %} — who cites whom. Directed edges from citing article to cited article.
  • {% if current == "article_cocit" %}Article Co-Citation{% else %}Article Co-Citation{% endif %} — which articles get cited together. Perceived kinship as the citing community treats it (built up by later readers).
  • {% if current == "author_cocit" %}Author Co-Citation{% else %}Author Co-Citation{% endif %} — which scholars get cited together. The same kinship signal aggregated up to author level.
  • {% if current == "coupling" %}Bibliographic Coupling{% else %}Bibliographic Coupling{% endif %} — which articles read the same things. Intellectual input as fixed at publication time (built into each article's reference list).

Citation is directed; the other three are undirected. Co-citation accumulates as the field reads; coupling is fixed when the article is written. The same article can be central in one view and peripheral in another — the disagreement is the analytic point.
