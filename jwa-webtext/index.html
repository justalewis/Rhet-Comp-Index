The Article
Discipline-Specific Bibliometrics in Rhetoric and Composition: Pinakes and the Infrastructure of Scholarly Visibility
A free, open, searchable index of Rhetoric and Composition scholarship, built so researchers can better visualize how sources cite one another over time. It gathers roughly 55,000 journal articles across fifty-five journals and more than 3,000 book and chapter records.

The Research Note
Toward a Code of Ethics for Scholarly Web Scraping in Rhetoric and Composition
A proposed code of ethics for the scholarly web scraping used to build bibliographic infrastructure in Writing Studies. Pinakes could not have been built without collecting metadata from publishers' sites, and that collection raised ethical questions the field has not yet addressed.
